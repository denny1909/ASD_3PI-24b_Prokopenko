# 🚀 Інструкція з запуску програми

## Вимоги
- ✅ Java Development Kit (JDK) 8 або новіше
- ✅ Windows/Linux/macOS
- ✅ Мінімум 512 MB RAM

## Швидкий старт (Windows)

### Варіант 1: Використання batch файлів

1. **Компіляція:**
   ```cmd
   compile.bat
   ```

2. **Запуск:**
   ```cmd
   run.bat
   ```

### Варіант 2: Ручна компіляція

1. **Компіляція:**
   ```cmd
   cd src
   javac *.java
   ```

2. **Запуск:**
   ```cmd
   java Main
   ```

## Швидкий старт (Linux/macOS)

1. **Компіляція:**
   ```bash
   cd src
   javac *.java
   ```

2. **Запуск:**
   ```bash
   java Main
   ```

## 📖 Як користуватися програмою

### Крок 1: Завантаження точок
1. Натисніть кнопку "📁 Завантажити файл"
2. Виберіть один з файлів:
   - `points.txt` (15 точок - швидко)
   - `points_50.txt` (50 точок - середньо)
   - `points_temp.txt` (якщо є)

### Крок 2: Вибір алгоритму
Виберіть один з двох доступних алгоритмів:

**🟢 Жадібний алгоритм (Рекомендовано для великих наборів):**
- ✅ Швидкий
- ✅ Підходить для великих наборів даних
- ⚠️ Може не знайти абсолютний оптимум

**🔵 Алгоритм з поверненням (Backtracking):**
- ✅ Знаходить оптимальне рішення
- ⚠️ Повільніший для великих наборів
- 💡 Краще працює на 15-30 точках

### Крок 3: Запуск обчислень
1. Натисніть "▶ Запустити алгоритм"
2. Спостерігайте за прогрес-баром (як у іграх!)
3. Дочекайтеся завершення

### Крок 4: Перегляд результатів
- **Ліва панель:** Графічне відображення трикутників
- **Права панель:** Детальна інформація
- **Час виконання:** Відображається внизу

### Крок 5: Збереження
1. Натисніть "💾 Зберегти результат"
2. Виберіть місце збереження
3. Файл буде збережено в текстовому форматі

## ❓ Часті питання (FAQ)

### Q: Програма не запускається
**A:** Переконайтесь що JDK встановлено і javac доступний у PATH

### Q: Помилка компіляції
**A:** Перевірте що всі .java файли знаходяться в папці src/

### Q: Довго виконується
**A:** 
- Виберіть Жадібний алгоритм для великих наборів
- Для Backtracking використовуйте не більше 30 точок

### Q: Не знаходить трикутники
**A:** 
- Переконайтесь що точки не колінеарні
- Потрібно щонайменше 3 точки

### Q: Як створити свій файл з точками?
**A:** Використовуйте формат:
```
Point 1: x=100; y=200;
Point 2: x=150; y=250;
...
```
