### ✅ Алгоритми

#### Алгоритм 1: Жадібний
- [x] Генерація всіх тупокутних трикутників O(n³)
- [x] Обчислення площі кожного трикутника
- [x] Сортування за площею O(m log m)
- [x] Жадібний вибір без спільних вершин O(m)
- [x] Загальна складність: O(n³ log n)
- [x] Просторова складність: O(n³)

#### Алгоритм 2: Backtracking
- [x] Генерація всіх тупокутних трикутників O(n³)
- [x] Рекурсивний перебір комбінацій
- [x] Pruning (відсікання неперспективних гілок)
- [x] Відстеження найкращого рішення
- [x] Відкат (backtracking)
- [x] Обмеження глибини рекурсії
- [x] Загальна складність: O(n³ × 2^(n/3))
- [x] Просторова складність: O(n³)
