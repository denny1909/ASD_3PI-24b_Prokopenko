# 🎨 Візуальний посібник програми

## Що таке тупокутний трикутник?

```
Тупокутний трикутник - це трикутник, у якого один з кутів більший за 90°

Приклад:
         A
        /|
       / |
      /  |
     /   |
    /    |
   /     |
  /   θ  |    θ > 90°  (тупий кут)
 /_______|
B         C

Математично: якщо BC² > AB² + AC², то кут A - тупий
```

### Крок 2: Генерація трикутників

```
Алгоритм перевіряє всі можливі комбінації з 3 точок:

Комбінація (1,2,3):
    3
   /|
  / |
 /  |
1───2

Перевірка:
✅ Валідний трикутник?     Так (не вироджений)
✅ Тупокутний?             Так (кут при вершині 3 > 90°)
➡️ Додаємо до списку кандидатів
```

### Крок 3: Вибір трикутників (Жадібний)

```
1. Сортуємо за площею:
   T1(5,10,13)  - площа: 850 ⭐⭐⭐⭐⭐
   T2(1,2,3)    - площа: 250 ⭐⭐
   T3(4,6,9)    - площа: 600 ⭐⭐⭐⭐
   T4(7,11,12)  - площа: 400 ⭐⭐⭐

2. Жадібно вибираємо:
   ✅ T1(5,10,13) - площа 850 → вибрано
      Використані точки: {5, 10, 13}
   
   ❌ T3(4,6,9) - має точку 9? Ні → перевіряємо далі
   ✅ T3(4,6,9) - вибрано
      Використані точки: {5, 10, 13, 4, 6, 9}
   
   ❌ T4(7,11,12) - має точку 11? Ні → перевіряємо далі
   ✅ T4(7,11,12) - вибрано
      Використані точки: {5, 10, 13, 4, 6, 9, 7, 11, 12}
   
   ❌ T2(1,2,3) - має спільні точки? Ні
   ✅ T2(1,2,3) - вибрано

Результат: 4 трикутники
```

## Структури даних у дії

## Оптимізації алгоритму

### Pruning в Backtracking:
```
Поточне рішення: [T1, T2, T3]     (3 трикутники)
Найкраще рішення: [T4, T5, T6, T7] (4 трикутники)

Залишилось точок: 6
Максимум можливих трикутників: 6/3 = 2

Чи можливо побити рекорд?
3 (поточні) + 2 (максимум) = 5 трикутників
5 > 4 ✅ - продовжуємо пошук

Якщо б було:
2 (поточні) + 2 (максимум) = 4
4 ≤ 4 ❌ - відсікаємо гілку (pruning)
```
